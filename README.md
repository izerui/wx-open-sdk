# 微信开放平台sdk (基于retrofit2)

# 在你的项目中使用:
## gralde:
第一步: 添加 JitPack repository 到你的 build.gradle 文件中

```java
allprojects {
		repositories {
			...
			maven { url "https://jitpack.io" }
		}
	}
```

第二步: 添加依赖

```java
dependencies {
	        compile 'com.github.izerui:wx-open-sdk:1.0-SNAPSHOT'
	}
```
